# 🚨 Security Incident Report

## Project Overview

This project documents the investigation of a simulated cybersecurity incident and outlines the steps taken to contain and mitigate the threat.

---

## Incident Summary

An employee reported suspicious login attempts on their Microsoft 365 account from an unfamiliar location.

---

## Investigation

- Reviewed login activity.
- Identified repeated failed login attempts.
- Detected a successful login from an unusual IP address.
- Confirmed the account owner did not initiate the login.

---

## Response Actions

- Reset the user's password.
- Enabled Multi-Factor Authentication (MFA).
- Revoked active sessions.
- Reported the incident to the security team.
- Continued monitoring for additional suspicious activity.

---

## Lessons Learned

- Strong passwords are essential.
- MFA significantly reduces account compromise.
- Continuous monitoring enables faster detection.
