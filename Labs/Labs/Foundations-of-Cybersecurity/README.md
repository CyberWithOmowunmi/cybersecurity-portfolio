# Foundations of Cybersecurity

## Overview

This project summarizes the knowledge and skills I gained while completing the first course of the Google Cybersecurity Professional Certificate.

## Learning Objectives

- Understand the role of cybersecurity in protecting organizations.
- Learn the CIA Triad (Confidentiality, Integrity, and Availability).
- Identify common cyber threats and attacks.
- Understand the responsibilities of a Cybersecurity Analyst.
- Explore security frameworks and best practices.

## Key Topics Covered

- Cybersecurity fundamentals
- The CIA Triad
- Threats, Risks, and Vulnerabilities
- Malware
- Phishing
- Ransomware
- Social Engineering
- Security Controls
- Security Ethics
- Security Frameworks

## Skills Demonstrated

- Security awareness
- Risk identification
- Threat analysis
- Basic incident response concepts
- Cybersecurity best practices

## Tools and Platforms

- Coursera
- Google Cybersecurity Professional Certificate
- GitHub

## Reflection

This course strengthened my understanding of cybersecurity fundamentals and the importance of protecting organizational assets. It also introduced the responsibilities of a cybersecurity analyst and reinforced the value of continuous learning in the cybersecurity field.
